# パンドラの人狼 — ゲームデザインドキュメント

**13人用・特殊ルール人狼 / 制作: ex-labs**

> 「死が、箱を開く。」
>
> 村はずれの祠に、開けてはならない箱が祀られていた。
> 人が死ぬたび、箱の蓋はわずかに軋み、中から災厄が這い出してくる。
> 箱の底に眠るのは、最後の希望——それとも、絶望か。

---

## 1. コンセプト

通常の人狼(汝は人狼なりや?)をベースに、**「パンドラの箱」** という第3の脅威を盤面の中央に置いた13人用の特殊人狼。

- **死が箱を開く**: 処刑・襲撃で死者が出るたび、翌朝「パンドラの箱」から災厄カードが1枚公開され、その日のルールが変化する。
- **箱の底には希望**: 山札のどこか(下寄り)に1枚だけ「希望」が眠っており、公開された瞬間すべての災厄が解除される。
- **第3陣営「パンドラ」**: 箱が開くことを望む少女が村に紛れており、災厄が十分に世へ放たれた状態でゲームが終わると、勝利を独り占めする。

通常人狼の「誰が狼か」という推理に、**「箱をこれ以上開けてよいのか」という村全体のリソース管理**と、**「死者を出さない日を作る」という新しい防衛目標**が重なるのが本作の核。護衛成功や処刑回避(引き分け)が単なる停滞ではなく「箱を止める」という積極的な戦術になる。

- プレイ人数: **13人 + GM 1人**
- プレイ時間: 約60〜90分
- 対象: 人狼経験者向け(初心者は経験者とペア推奨)

## 2. 配役(13人)

| 陣営 | 役職 | 人数 | 備考 |
|---|---|---|---|
| 村人陣営 | 占い師 | 1 | 毎夜1人を占い「人狼か否か」を知る |
| 村人陣営 | 霊媒師 | 1 | 処刑された者が人狼だったか知る |
| 村人陣営 | 騎士 | 1 | 毎夜1人を護衛(連続護衛可・自己護衛不可) |
| 村人陣営 | **箱の番人** ★新役職 | 1 | 箱の公開を封印できる(2回まで) |
| 村人陣営 | 村人 | 4 | 能力なし |
| 人狼陣営 | 人狼 | 3 | 毎夜1人を襲撃。互いを認識 |
| 人狼陣営 | 狂人 | 1 | 人間だが人狼の勝利が自分の勝利 |
| 第3陣営 | **パンドラ** ★新役職 | 1 | 単独勝利を狙う禁忌の少女 |

- 村人陣営 8 / 人狼陣営 4 / 第3陣営 1
- 占い結果: 「人狼」= 人狼のみ。狂人・パンドラ・箱の番人はすべて「人狼ではない」。
- 霊媒結果も同様(人狼か否かのみ)。

## 3. パンドラの箱(中央システム)

### 3-1. 箱の中身

**災厄カード6枚 + 希望カード1枚 = 計7枚** の山札を作る。

組み方(GMが全員の前で行うと緊張感が出る):

1. 災厄6枚を裏向きでシャッフルする。
2. 上から4枚を取り、山の上部とする。
3. 残り災厄2枚に希望1枚を混ぜてシャッフルし、山の下部とする。

→ **最初の4枚は必ず災厄。希望は下から1〜3枚目のどこか**に眠る。「箱の底には希望が残った」という神話をルールで再現しつつ、終盤は「次の1枚が希望かもしれない」というギャンブルが生まれる。

### 3-2. 箱が開く条件

- 毎朝(2日目以降)、**前日の昼〜夜に死者が1人でも出ていた場合**、GMは箱の一番上のカードを1枚公開する。
- 死者が0人の日(護衛成功+処刑引き分けなど)は箱は開かない。
- 公開された災厄は **その日1日だけ** 有効。翌朝には解除される(重複しない)。
- 残り枚数は常に公開情報。GMは毎朝「箱の中には残り◯枚」と宣言する。

### 3-3. 災厄カード(6種)

| 災厄 | 効果(その日のみ) |
|---|---|
| 🌫 **沈黙の災厄** | 昼の議論時間が半分になる |
| 🌑 **暗闇の災厄** | 投票が無記名になる(GMが集計し、誰が誰に入れたか公開しない) |
| 👁 **盲目の災厄** | 今夜の占いは無効(結果が返らない) |
| 🕯 **忘却の災厄** | 本日の処刑者の霊媒結果は永遠に得られない |
| ⛈ **嵐の災厄** | 今夜、騎士は護衛できない |
| 🍽 **飢餓の災厄** | 本日の処刑は2名(投票上位2名。同数は決選投票) |

災厄はおおむね村側の情報・防御を削るが、「飢餓」だけは狼を2枚抜きうる諸刃の剣。どの災厄がまだ箱に残っているかを数えるカウンティングも重要な技術になる。

### 3-4. 希望カード「エルピス」

公開された瞬間、以下がすべて起こる:

1. **箱は永遠に閉じる。** 以後、災厄は一切発生しない。
2. **その日の処刑は行われない**(慈悲の一日。議論と夜時間は通常どおり)。
3. **パンドラの単独勝利条件が消滅し、パンドラは村人陣営に転向する**(勝利条件が村人陣営と同じになる。占い結果は変わらず「人狼ではない」)。

パンドラにとって希望は最大の敵であり、村にとっては起死回生の切り札。終盤「あえて死者を出して箱を掘り、希望を引きに行く」という逆転の択が村側に生まれる。

## 4. 新役職の詳細

### 4-1. 箱の番人(村人陣営)

祠を代々守ってきた一族の末裔。箱の悲鳴を聞くことができる。

- **封印(2回まで)**: 夜に「封印」を宣言すると、翌朝のカード公開が1回スキップされる(死者が出ていても箱は開かない)。使用回数はゲームを通じて2回まで。
- **軋みの感知**: パンドラが能力「こじ開け」を使った夜、番人はGMから「箱が悲鳴を上げた」と通知される(誰が使ったかは分からない)。
- 占い・霊媒結果は「人狼ではない」。

デザイン意図: パンドラと災厄に対する村側の唯一の直接カウンター。ただし2回しか使えないため「どの災厄を止めるか」「希望を掘る終盤には温存すべきか」という判断が問われる。狼にとっては占い師と並ぶ襲撃優先目標。

### 4-2. パンドラ(第3陣営)

好奇心に魅入られた少女。箱が世界に開かれることだけを望んでいる。

- **勝利条件**: 村人陣営または人狼陣営の勝利が確定した時点で、①パンドラが生存しており、②災厄が4枚以上公開済みなら、**その勝利を上書きしてパンドラの単独勝利**となる(妖狐型の漁夫勝利)。
- **箱の加護**: 人狼の襲撃では死なない(GMは「襲撃失敗」とだけ告げる。狼側は護衛成功と区別できない)。処刑では死ぬ。
- **こじ開け(1回だけ)**: 夜に使用すると、翌朝は死者の有無にかかわらずカードが**2枚**公開される(番人の封印と同夜にぶつかった場合、相殺して1枚公開)。
- 占い・霊媒結果は「人狼ではない」。

デザイン意図: パンドラは「死者が出続けること」を望むため、序盤は狼の襲撃を放置し、疑いを混ぜ返して処刑を促す動きになる。一方で自分は処刑されてはいけないので、信用を稼ぎながら村を疑心暗鬼に保つ、狂人とも狼とも違う独特の立ち回りが要求される。襲撃無効は生存保険だが、狼の襲撃が連続で失敗すると「護衛が神がかっているか、パンドラがいるか」という情報にもなる。

## 5. ゲームの流れ

```
【0日目・夜】 役職配布 → 人狼の相互確認(初日襲撃なし) → 箱の山札を全員の前で構築
【1日目・昼】 議論 → 投票 → 処刑(1名)
【1日目・夜】 占い / 護衛 / 封印・こじ開け宣言 / 襲撃
【2日目・朝】 死者発表 → 死者がいれば箱が開く(災厄公開) → 残り枚数宣言
【2日目・昼】 災厄の効果下で議論 → 投票 → 処刑
   …以下、朝(開封判定)→昼→夜 を繰り返す…
```

### 勝利条件まとめ

| 陣営 | 条件 |
|---|---|
| 村人陣営 | 人狼を全滅させる |
| 人狼陣営 | 人狼の数が村人陣営の生存者数以上になる |
| パンドラ | 上記いずれかの成立時に、生存 かつ 災厄4枚以上公開済み(単独勝利・上書き) |

- 勝敗判定時、パンドラは村人側の頭数に数えない(狼2・村2・パンドラ1なら狼勝利が確定し、そこでパンドラ条件を判定)。
- 希望公開後は、転向したパンドラは村人陣営として頭数に数える。

## 6. GM運用ガイド

- **毎朝の定型句**: 「昨日、村は◯人を失いました。箱が軋みます——(公開)——本日の災厄は『◯◯』。箱の中には残り◯枚」。死者0の朝は「箱は沈黙しています」。演出がこのゲームの半分。
- **襲撃失敗の告知は理由を伏せる**: 護衛成功・パンドラ襲撃・(封印とは無関係)いずれでも「今朝、死者はいませんでした」で統一。
- **封印とこじ開けの同夜処理**: 相殺して1枚公開。番人には「こじ開けを感知した」通知を必ず行う。
- **暗闇の災厄の日**: 投票用紙かチャットのDM投票に切り替える。集計は合計票数のみ発表。
- **飢餓の災厄の日**: 処刑2名によりゲームが一気に終盤化する。残り人数が少ない場合(生存7人以下)は「処刑1名+翌朝もう1枚公開」に差し替えてよい(バリアントB参照)。
- **タイムテーブル目安**: 議論 昼5〜7分(沈黙の災厄時は半分)、夜2分。全体で6〜7日目に決着する想定。

## 7. 戦略の見どころ(ブリーフィングで紹介したい読み筋)

- **村側**: 「死者を出さない日」を意図的に作れるのは騎士の護衛成功だけではない。あえて処刑を引き分けに持ち込み箱を止める、という通常人狼には無い択がある。ただし処刑をサボれば狼が減らない。ジレンマの設計。
- **狼側**: 災厄の多くは村の情報役職を潰すため、狼は基本的に箱が開くほど有利。ただし4枚を超えるとパンドラの単独勝利圏に入るため、**終盤は狼がパンドラ探しを始める**という奇妙な共闘構造が生まれる。
- **パンドラ**: 序盤は「よく喋る善良な村人」を演じて処刑を回し、災厄4枚が見えたら潜伏に切り替える。こじ開けの使い所(死者0の朝を強引に開ける)が腕の見せ所。
- **箱の番人**: カミングアウトすれば襲撃筆頭。潜伏したまま「なぜか箱が開かなかった朝」を作ると、村に匿名の安心を、狼とパンドラに焦りを与えられる。

## 8. バリアントルール

- **A. 深淵の箱(上級)**: 災厄を7枚に増やし(追加: 「猜疑の災厄」= 本日、役職者は能力に関する発言をしてはならない)、パンドラの条件を5枚に引き上げる。
- **B. 少人数終盤補正**: 生存7人以下で「飢餓の災厄」が出た場合、処刑1名+翌朝追加で1枚公開に差し替え。
- **C. 希望の代弁者**: 希望公開時、直前までの死者から1名がGM指名で「最後の言葉」を1分だけ全体に残せる(推理情報の直接開示は禁止。観戦者を巻き込む演出用)。
- **D. 短縮版(60分)**: 災厄を4枚+希望1枚(希望は下から1〜2枚目)、パンドラ条件を3枚に。

## 9. デザインノート(バランス根拠)

- ベースは標準的な13人村(狼3・狂1・占・霊・騎・村7)。村人1枠をパンドラに、もう1枠を箱の番人に置き換えた。番人は村の実質戦力なので村側の地力は標準からほぼ落ちていない。
- 災厄6枚のうち5枚は村側の情報・防御を削る効果で、狼有利に寄る。その補正として (1) 死者0で箱が止まる仕様(護衛の価値が標準より高い)、(2) 番人の封印2回、(3) 希望による災厄全解除+処刑スキップ、の3つの村側リソースを用意した。
- パンドラの「災厄4枚」ラインは、平均的な進行(ほぼ毎日死者1)で4〜5日目に到達する。つまり中盤以降は常にパンドラ勝利が視野に入り、終盤の緊張を担保する。早すぎる単独勝利はパンドラ自身の処刑リスク(全員が探し始める)で抑制される。
- 「襲撃無効」は妖狐の系譜だが、占いで死なない代わりに勝利条件が受動的(自力では勝てず、他陣営の決着に相乗りする)ため、妖狐より村の負担は軽い。

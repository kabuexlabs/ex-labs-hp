# 東急電鉄 沿線イマーシブ体験企画 — 全体概要

**提案:株式会社エクスペリエンスラボ**
**作成日:2026-07-04(社内検討用ドラフト)**

---

## 1. 本提案の骨子

東急線の**通常営業時間中の電車と駅**をそのまま舞台に使い、参加者が「日常の乗客のふり」をしながら物語の当事者になるイマーシブ(没入型)体験を提案する。

核となる体験は次の4つ。

1. **隣の乗客が、突然話しかけてくる**
   演者が一般客として先回りして乗車しており、参加者の隣で"ごく自然に"接触してくる。「日常が物語に侵食される」瞬間を体験の入口にする。
2. **電車に乗ること自体がゲームメカニクスになっている**
   実在のダイヤ・列車番号・乗り継ぎが謎と直結する。「この電車に乗らなければ進めない」「乗り遅れ=失敗」という設計で、移動が消化ではなく緊張になる。
3. **駅構内の"入れない場所"に入れる**
   駅事務室、券売機の裏側、社員用通路、遺失物保管庫、検車区など、乗客が絶対に立ち入れないエリアが物語の到達点になる。
4. **参加者に常に「焦り」がある**
   実ダイヤを締切として使う。ゲーム内タイマーではなく「14:32発の各停に乗れなければ終わり」という現実の時計が迫ってくる。

## 2. なぜ「営業時間中」で成立するのか(設計思想)

貸切や終電後ではなく、あえて日中の営業時間中に実施する。成立させる鍵は、**「目立ってはいけない」を物語上の掟にする**こと。

- プラン1(爆破阻止)では「パニックを起こさず秘密裏に任務を遂行せよ」が任務条件そのもの。
- プラン2(0番線)では「気配を悟られると"あの人"が消えてしまう」が世界観のルール。

つまり、**参加者が一般客と見分けがつかない状態を保つことが、ゲーム上の勝利条件と一致する**。運営上の制約(騒がない・走らない・機器に触れない)が、そのまま没入を深める演出になる。これが本企画の最大の発明である。

具体的な運用は以下。

- 参加者は私服・2〜4名1組。スマートフォン(専用Webアプリ/LINE)+片耳イヤホンで指示を受ける。
- 組ごとに15〜20分間隔のスタート時刻をずらし、同一車両・同一箇所に集中させない。
- 駅裏側への入場は「改札窓口で合言葉を告げる → 係員(演者または研修済み駅係員)が案内」という形式。傍目には「駅員に案内される客」にしか見えない。
- 演者はすべて一般乗客・駅係員・清掃員・工事関係者などの「駅に居て当然の人」に擬態する。

## 3. 推奨路線

**池上線・東急多摩川線を主舞台、大井町線を接続路線**とする。

| 観点 | 理由 |
|---|---|
| 車両 | 3両ワンマン運転。車内で参加者と演者の位置関係をコントロールしやすい |
| 混雑 | 日中は閑散しており、隣に座る・小声で話す演出が自然に成立する |
| 運行間隔 | 日中おおむね6〜8分間隔。「1本逃す」ペナルティが重すぎず軽すぎない |
| 駅 | 戸越銀座・御嶽山など小規模駅が多く、裏側動線が短い。雪が谷大塚には検車区が隣接 |
| 乗り継ぎ | 旗の台(大井町線)、蒲田(多摩川線)、大岡山・自由が丘へ拡張可能で「乗り継ぐ意味」を作りやすい |
| 沿線施策 | 戸越銀座商店街など回遊施策と接続でき、東急側の日中需要創出・沿線ブランディングに直結 |

## 5. 東急電鉄側のメリット(交渉材料)

- **日中閑散時間帯(10時〜16時)の乗車需要創出**。参加者は1回の体験で最低4〜6回乗車する。
- 参加条件に東急線ワンデーパス等を組み込み、**企画乗車券の販売増**に直結。
- 「東急の駅の裏側に入れる」は鉄道会社にしか出せない体験価値であり、**沿線ブランディング・ファン化施策**として機能する。
- 戸越銀座・蒲田等の**商店街・沿線商業施設への送客**を体験内に組み込める。

## 6. 立入エリアの想定(共通)

交渉対象として現実的な範囲に絞る。

- ○ 駅事務室・駅長室(小道具設置・演者配置の主舞台)
- ○ 改札窓口の内側/券売機背面の機器スペース(見るだけでも強烈な非日常)
- ○ 社員用通路・階段、使用していない旧改札・旧売店区画
- ○ 遺失物の保管スペース(プラン2の心臓部)
- ○ 雪が谷検車区の見学可能エリア(クライマックス候補)
- × 線路内・ホーム下・電気室・信号扱い所は**対象外**(そもそも要望しない)

## 7. 事業規模ざっくり(共通試算)

- 体験時間:約120分/参加費:5,500円〜6,600円(税込)前後
- 1日あたり:15分間隔×6時間 → 最大24組(1組2〜4名、日次60〜90名)
- 会期:週末中心の2〜3ヶ月ロングラン想定
- 収入源:チケット/企画乗車券セット販売/グッズ(プラン2は「忘れ物」モチーフのグッズ親和性が非常に高い)/沿線店舗タイアップ

# 形見の人狼 — ゲームデザインドキュメント

**13人用・特殊ルール人狼 / 制作: ex-labs**

> 「疑うべきは人ではなく、系譜。」
>
> この村の力は、血ではなく形見に宿る。
> 占いの水晶も、守りの盾も、持ち主が死ねば誰かの手に渡る。
> 水晶は嘘をつかない。だが、水晶を握るその手は——。

---

## 1. コンセプト — 人狼の構造のどこを壊したか

通常人狼の骨格は、**「役職(能力)と陣営(正体)が、1枚のカードに溶接されている」**ことで成り立っている。占い師であることは村人陣営であることを意味し、だからこそCO・対抗CO・ローラーという定跡が生まれる。

本作はこの溶接を切断する。

1. **能力と陣営を別のカードに分離する。** 能力は「形見(かたみ)」という持ち運べる道具になり、陣営だけが人に固定される。
2. **能力の所在は常に全公開、陣営だけが秘匿。** 「占い師が誰か」は初日から全員が知っている。分からないのは「その占い師が狼かどうか」だけ。通常人狼と情報構造が真逆になる。
3. **死は退場であると同時に「相続」である。** 死者は形見を、自分が選んだ生存者に公開で遺す。能力は殺しても消えず、絆を渡って旅を続ける。

この3点により、人狼の中心的な問い自体が変わる。

> 通常人狼: 「占い師を騙っているのは誰か?」
> 形見の人狼: 「本物の水晶を握った人間は、本物の報告をしているか?」

騙り(偽CO)という概念が消滅し、代わりに**「形見の系譜(誰の手を、どの順に渡ってきたか)を読む」**という新しい推理が生まれる。水晶が3人の手を渡ったなら、報告の食い違いはどの代で混入したのか——推理の対象が「人」から「相続の歴史」に移る。

- プレイ人数: **13人 + GM 1人**
- プレイ時間: 約60〜90分
- 対象: 人狼経験者ほど衝撃が大きい(定跡が全部使えなくなるため)。ルール量自体は通常人狼+形見だけなので初心者も入れる

## 2. 配役(13人)— 陣営カード

陣営カードは通常どおり秘匿で配布。**能力を持つ役職は存在しない。** 能力はすべて形見側にある。

| 陣営 | 人数 | 備考 |
|---|---|---|
| 村人陣営(村人) | 9 | 全員素の村人。能力は形見で得る |
| 人狼陣営(人狼) | 3 | 毎夜1人を襲撃。互いを認識 |
| 人狼陣営(狂人) | 1 | 人間だが人狼の勝利が自分の勝利。狼が誰かは知らない |

- 勝敗判定は通常どおり(村: 狼全滅 / 狼: 生存狼数 ≥ 生存人間数。狂人は人間側に数える)。
- 死者は退場する(遺言と形見分けが最後の仕事)。

## 3. 形見(中央システム)

### 3-1. 4つの形見

ゲーム開始時、GMが全員の前でくじを引き、**陣営に関係なくランダムな4人**に1つずつ公開で渡す。初日から狼が水晶を握っているかもしれない——それがこのゲームである。

| 形見 | 効果 |
|---|---|
| 🔮 **占いの水晶** | 毎夜、1人を占う。結果(人狼か否か)は保持者だけに伝えられる |
| 🛡 **守りの盾** | 毎夜、1人を護衛する(自分は護衛できない。連続護衛可) |
| 🔔 **霊媒の鈴** | 処刑者が人狼だったか否かを、保持者だけが知る |
| ⚖ **裁きの天秤** | 保持者の処刑投票は2票として数える(常時・公開) |

- **形見の所在は常に公開情報。** GMは毎朝「水晶は◯◯さん、盾は△△さん……」と所在を読み上げる。
- **結果は保持者だけのもの。** 水晶や鈴が何を告げたかを村に報告する義務はなく、嘘をついてもよい。**形見は真実を告げるが、口を持たない。** 語るのは常に、陣営を隠した人間である。
- 形見は狼が持っても完全に機能する(狼が水晶で占えば真の結果を得る。仲間の位置確認や、精度の高い嘘の材料になる)。
- 1人で複数の形見を持ってよい(相続で集中しうる)。持てば持つほど夜は忙しく、昼は狙われる。

### 3-2. 形見分け(死=相続)

プレイヤーが死んだとき、持っていた形見は必ず相続される。

- **処刑された者**: 遺言(30秒)ののち、形見を1つずつ、任意の生存者に**全員の前で**手渡して退場する(複数持ちなら別々の相手に分けてもよい)。
- **襲撃された者**: 翌朝、死亡発表の直後に同様に形見分けを行ってから退場する(夜のうちに死んでいるが、「今際の際の遺志」として本人が渡し先を決める)。
- 渡し先の拒否はできない。相続の連鎖に上限はない。

**形見分けは、このゲーム最大の情報イベントである。** 死にゆく者が誰を選ぶかは「私はこの人を信じて死ぬ」という最後の意思表示であり、全員がそれを見ている。狼にとっては、殺しの成果が「信頼された誰か」を強化してしまうという構造的な逆風になる。

### 3-3. 構造が生む力学

- **殺しても、力は消えない。** 水晶の保持者を襲撃しても水晶は次の手に渡るだけ。狼が本当に破壊すべきは形見ではなく「信頼の網」であり、そのためには自分たちが相続線に潜り込む(=信頼を稼いで形見を受け取る)しかない。
- **形見分けは信頼グラフを可視化する。** 誰が誰に遺したかの履歴は全公開。村はこの系譜を読んで「相続線が狼に汚染された代」を特定しようとし、狼は汚染を1世代でも遅く発覚させようとする。
- **報告の真偽は系譜でしか検証できない。** 「3日目までの水晶の報告」と「4日目以降の報告」が矛盾したら、疑うべきは3日目の保持者か4日目の保持者か、それとも占われた側の芝居か。
- **天秤だけは嘘がつけない形見。** 効果が公開・自動のため、純粋な政治力として相続される。終盤の2票はキングメーカーであり、天秤の相続線は毎回熾烈な議論になる。

## 4. ゲームの流れ

```
【0日目】   陣営カード配布 → 人狼の相互確認(初日襲撃なし)
          → 全員の前でくじ引き、形見4つを公開で配布
【1日目・昼】 議論 → 投票 → 処刑 → 遺言・形見分け(公開)
【1日目・夜】 盾の護衛 → 人狼の襲撃 → 水晶の占い
【2日目・朝】 死亡発表 → 犠牲者の形見分け(公開) → 形見の所在の読み上げ
【2日目・昼】 議論(鈴の保持者は前日の処刑者の白黒を知っている) → 投票 → 処刑…
   …以下繰り返し…
```

- 夜の処理順: 盾 → 襲撃 → 水晶(水晶の保持者が襲撃されても、その夜の占い結果は死の間際に得ており、形見分けの際に**口頭で言い残してよい**——真偽はやはり保証されない)。
- 形見を持たない者の夜は通常人狼の村人と同じ(役職欠けの心細さも含めて)。

## 5. GM運用ガイド

- **形見は物理的に持たせる**(水晶・盾・鈴・天秤のプロップを製作)。所在公開のルールが自然に守られ、卓上の風景そのものが盤面になる。相続の瞬間、手から手へ渡る演出が最大の見せ場。
- **毎朝の所在読み上げ**を儀式化する: 「水晶は◯◯の手に。盾は△△の手に。——鈴は、昨夜その主を喪いました」。
- **結果の伝達**は保持者にのみ、紙またはハンドサインで。周囲に覗かれない運用を徹底する(結果が公的に証明できないことがゲームの根幹)。
- **形見分けの迷い時間**は30秒まで。決められない場合はGMがくじで決める(「遺志なき形見は、風に流れる」)。
- **同数処理**: 処刑投票の同数は決選投票→なお同数なら処刑なし(形見分けも発生しない)。
- **タイムテーブル目安**: 昼5〜7分、夜2分。6〜7日目決着想定。

## 6. 戦略の見どころ

- **村側**: 形見を「確定白に集める」のが理想だが、このゲームに確定白はほぼ存在しない。次善は**相続線を分散させ、1つの汚染で全部を失わない**こと。死が見えたら(処刑されそうになったら)、自分の疑い筋を遺言で語り、形見分けでそれを裏書きする——言葉と相続先が一致しているかも読み筋になる。
- **狼側**: 開幕の理想は「初期くじで形見を引く」こと。引けなければ、**中盤までに1本の相続線に潜り込む**のが生命線。水晶を得たら真の結果で正確な嘘をつけるが、報告が正確すぎる水晶はかえって疑われる。また「あえて形見保持者を襲撃しない」ことで、村の相続イベントを起こさせず情報を凍結する遅滞戦術もある。
- **狂人**: 形見の受け取り競争における最強のダークホース。人間なので水晶にも鈴にも「人狼ではない」と出る。信頼を稼いで水晶を相続し、堂々と偽の結果を報告し続ける狂人は、通常人狼の偽COより遥かに破壊的。
- **全員**: 「もし今夜死んだら、誰に何を遺すか」を毎晩考えることになる。この問いはそのまま「自分は誰を信じているのか」であり、13人の信頼関係が形見の動線として物理的に卓上に現れる。

## 7. バリアントルール

- **A. 贋作(上級)**: 水晶を2つ入れ、片方は贋作(結果がランダム)。**どちらが真作かは保持者にも分からない。** 系譜の推理に「そもそも初代から狂っていた可能性」が加わる。
- **B. 相続税**: 1人が持てる形見は2つまで。3つ目を相続する場合、手持ちの1つをその場で他の誰かに譲る(権力集中の抑制)。
- **C. 呪いの形見(第3勢力入り・14人)**: 5つ目の形見「黒鏡」を追加。保持しているだけで毎夜、自分の陣営が「人狼」として占われるようになる呪具。押し付け合いと、あえて抱え込む献身が生まれる。
- **D. 遺言状**: 各自、ゲーム開始時に相続先を書いた遺言状をGMに預けておき、夜死んだ場合はそれに従う(朝の形見分けの代わりに開封)。書き直しは1日1回夜のみ。誰がいつ書き直したかは公開される——書き直しの履歴がまた読み筋になる。

## 8. デザインノート

### 8-1. なぜこれが「構造の改変」なのか

- **1作目(パンドラ)**は環境ルールの変動、**2作目(彼岸)**は死者の再利用で、どちらも人狼の骨格(役職=陣営の溶接、CO文化、死=情報の消滅)には触れていなかった。本作は骨格側の3点を直接書き換えている: ①能力と陣営の分離 ②公開/秘匿の反転 ③死の意味の書き換え(退場イベント→意思決定イベント)。
- 副次効果として、通常人狼の主要な不満点が構造の内側から解決されている。「役職を引けなかった村人が暇」→ 誰でも相続で当事者になりうる。「初日処刑が空虚」→ 初日から形見分けという重い意思決定がある。「騙りローラーの作業感」→ 騙り自体が存在しない。**サブシステムを足して直したのではなく、骨格を変えたら勝手に直った**、という設計になっていることが重要。

### 8-2. バランス根拠

- 陣営比は 人間10(村9+狂1) : 狼3。通常13人村から役職を全て抜いた代わりに形見4つを入れており、形見が全て村人の手にあれば村側は標準よりやや強い。その補正が「初期配布は陣営無関係のランダム」(期待値で形見4つのうち約1.2個が狼陣営スタート)と、狂人の相続参入。テストプレイではまず標準配分で回し、村が勝ちすぎる場合はバリアントA(贋作)を、狼が勝ちすぎる場合は初期配布を「人間のみ」に変えて調整する。
- 天秤(2票)は情報系でない形見をあえて1枚入れることで、「形見=情報源」と単純化させないための異物。終盤のPARITY計算を壊す装置でもある(狼2村3でも天秤の位置次第で詰みが変わる)。

### 8-3. 併せて検討した別の「構造改変軸」(ボツ案・次回作候補)

- **王座の人狼(処刑の民主制を解体)**: 毎日、多数決で処刑するのではなく「執政官」を選挙し、執政官が独断で1人を処刑する。議論が「吊り先」ではなく「権力の委任先」を巡るものに変わる。既存作(Secret Hitler)との類似が強く今回は見送り。
- **渡り火の人狼(陣営の固定を解体)**: 「人狼」を人ではなく夜ごとに隣へ移りうる呪いとして扱う。昨日の狼が今日は村人。推理対象が「誰か」から「呪いの現在地」になる。構造的には最も過激だが、追跡不能感がストレスになりうるためプロトタイプ検証が必要と判断。
